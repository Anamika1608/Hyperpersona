# Design Inspiration

Add screenshots, recordings, product references, and visual moodboards here.

## Intended use

- local inspiration for layout, motion, hierarchy, and styling decisions
- a staging area before formal token refinement
- a shared folder for references tied to specific routes or modules

## Important note

`extract-design-system` should only be used when there is a public website URL to analyze. Local images and videos in this folder are for manual inspiration, not direct token extraction with that skill.
